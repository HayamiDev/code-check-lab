# code-check-lab

コードレビュー練習問題をAI生成する

## ローカル実行

### 必要な環境

- Node.js 20以上

### セットアップ

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで http://localhost:5173 を開く

### ビルド

```bash
npm run build
```

`dist`フォルダに成果物が出力される

### プレビュー

```bash
npm run preview
```

ビルド成果物をローカルで確認できる
