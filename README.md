# code-check-lab
Code Check Labは、AIが生成する実践的なコードレビュー問題を通じて、エンジニアのレビュー能力を向上させるためのトレーニングアプリケーションです。

## 🚀 主な機能

### 1. AIによるパーソナライズされた問題生成
- **多言語対応**: Kotlin, Swift, TypeScript, Rust, Go, Pythonなど、主要な言語から選択可能。
- **難易度調整**: 初級（Lv.1）から上級（Lv.10）まで、自身のスキルに合わせた難易度で問題を生成。
- **実践的なシナリオ**: セキュリティホール、パフォーマンス低下、可読性の欠如など、現場で直面するリアルな欠陥を含むコードを生成。

### 2. 没入感のあるプレミアムUI体験
- **IDEライクなインターフェース**: JetBrains Monoフォントやシンタックスハイライトを採用し、実務に近い環境を再現。
- **ダイナミックな演出**: Framer Motionによる滑らかな画面遷移、高得点時のConfetti（紙吹雪）、スキャンエフェクトなど、触っていて楽しいインタラクション。
- **レイヤード・ダークモード**: 深みのあるグラデーションとグラスモーフィズム（すりガラス効果）を取り入れた、目に優しく洗練されたデザイン。

### 3. AIによる精密な自動採点 & フィードバック
- **個別評価**: 指摘の一つひとつに対して、AIが「妥当性」と「説明の十分さ」を判定。
- **具体的アドバイス**: なぜその指摘が良いのか、あるいは何が足りなかったのかを詳細に解説。
- **成長の可視化**: 過去のスコア推移をグラフで表示し、スキルアップを実感できます。

## 🛠 技術スタック

このプロジェクトは、最新のWebテクノロジーとAIサービスを組み合わせて構築されています。

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS (with Custom Glassmorphism System)
- **Animation**: Framer Motion, Canvas Confetti
- **Icons**: Lucide React
- **Language**: JavaScript (ESModules)
- **AI Integration**: Anthropic Claude 3.5 Sonnet (via Cloudflare Workers)

## 🔄 学習のフロー

1.  **Setup**: 言語と難易度を設定し、セッションを開始。
2.  **Review**: 「レビュー対象コード」を精査し、バグや改善点を特定。
3.  **Evaluate**: AIレビュアーに回答を提出。即座に採点が行われます。
4.  **Feedback**: スコア、AIからの総評、個別の解説を確認してスキルを吸収。

## 💻 ローカル実行

### 必要な環境
- Node.js 20以上

### セットアップ & 起動

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

`dist`フォルダに本番用ファイルが出力されます。

## ⚠️ API設定について

本アプリケーションは、バックエンドとして Cloudflare Workers 上のプロキシAPIを使用しています。
ローカル環境で動作させる場合でも、デフォルトで設定されているエンドポイント (`src/api/claude.js`) がプロキシ経由でAIサービスに接続するため、**クライアント側にAPIキーを設定する必要はありません。**

※ 独自のAPIキーを使用したい場合は、`src/api/claude.js` 内のエンドポイントとヘッダー設定を変更してください。
